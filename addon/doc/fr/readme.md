## Description
**Dictionary Display Lite** est votre purificateur vocal personnel pour les boîtes de dialogue de dictionnaire de NVDA. Vous connaissez cette annonce interminable chaque fois que vous parcourez une liste d'entrées ?

`Motif : OBS ; Remplacer par : O B S ; casse : désactivé ; Type : Expression régulière`

Avec **Dictionary Display Lite**, vous n'entendrez instantanément que l'essentiel :

`OBS O B S désactivé Expression régulière`

Fini les « Motif : », « Remplacer par : », « casse : » ou « Type : ». Il fonctionne silencieusement en arrière-plan, rendant votre navigation plus rapide et agréable.

> **Zéro configuration :** Installez l'extension, ouvrez n'importe quel dictionnaire et placez le focus sur un élément. NVDA prononcera automatiquement la version condensée.

## Pourquoi vous allez l'adorer
Si vous gérez de nombreuses entrées, vous savez à quel point les annonces complètes sont répétitives. **Dictionary Display Lite** filtre tout cela pour vous. Il ne laisse que le motif, son remplacement, le réglage de la casse et le type de règle.

C'est comme avoir un assistant personnel qui sait exactement quoi dire. Chaque annonce est précise et directe.

## Ce qu'il supprime
* **« Motif : »** – vous savez déjà qu'il s'agit d'un motif.
* **« Remplacer par : »** – la deuxième partie est évidemment le remplacement.
* **« casse : »** – vous entendrez « activé » ou « désactivé ».
* **« Type : »** – vous indique le type de règle.
* Tous les points-virgules et espaces superflus.

## Gardez toujours le contrôle
Le texte détaillé original n'est pas perdu. Il est seulement masqué de la parole. Pour tout entendre à nouveau :
* Appuyez sur **NVDA+Tab** pour lire l'objet avec tous les détails.
* Utilisez les commandes de revue d'objet (**NVDA+pavé numérique**) pour explorer le texte brut.
